常见问题

您可能关心的问题

授权是按服务器还是按项目计算？
专业版授权按单台服务器计算。每台运行智域 WAF 的服务器需要独立的授权码。如需多台服务器部署，可选择企业定制方案获得批量折扣和不限数量授权。
授权到期后会怎样？数据会丢失吗？
授权到期后，专业版功能将进入 7 天宽限期，宽限期内功能仍可正常使用。宽限期结束后自动降级为社区版，您的所有配置、规则和日志数据完全保留，规则引擎等基础功能继续正常运行，不会丢失任何数据。
如何更换服务器？迁移复杂吗？
授权码绑定机器指纹。如需迁移，在旧服务器的控制台「设置 → 授权」页面解除绑定，然后在新服务器上重新激活即可。如果旧服务器已无法访问，联系技术支持我们会协助处理，整个过程不超过 5 分钟。
开源版和专业版的代码有区别吗？
社区版代码完全开源，功能没有任何阉割。专业版通过在线授权中心验证 License Key 解锁高级功能模块，核心代码架构完全一致。我们不会通过闭源来制造功能差异，所有检测引擎和规则在社区版中均可使用。
支持哪些付款方式？
支持支付宝、微信支付、银行转账。企业客户可对公转账并开具增值税普通发票或专用发票。具体付款方式将在购买邮件中详细说明。
离线/内网环境能否使用专业版？
支持。专业版支持离线授权模式：首次激活需要联网验证，验证通过后授权 token 有效期为 90 天，期间完全无需联网。到期前系统会自动提示续期验证，只需短暂联网即可续期。
购买后可以升级方案吗？
可以。从社区版升级专业版，或从专业版升级企业定制，均可随时操作。升级时按照剩余授权期限折算差价，不会重复收费。
